# Matemática Financeira - Ensino Médio

Site em HTML com 10 aulas de Matemática Financeira para o Ensino Médio.

## Conteúdo
O material está organizado em:
- página principal com navegação para as aulas
- 10 páginas individuais, uma para cada aula
- explicação teórica
- exemplos resolvidos
- exercícios de múltipla escolha
- exercícios discursivos

## Temas abordados
1. Porcentagem: conceito e cálculo básico
2. Acréscimo e desconto simples
3. Aumento percentual e redução percentual
4. Orçamento pessoal e familiar
5. Consumo consciente e pesquisa de preços
6. Inflação e poder de compra
7. Juros simples
8. Juros compostos
9. Comparação entre juros simples e compostos
10. Endividamento, planejamento e revisão geral
